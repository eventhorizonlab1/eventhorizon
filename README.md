# Event Horizon

Event Horizon is a static website for a French-language news source focused on the European space industry. It is built with HTML, Tailwind CSS, and Alpine.js.

## Project Structure

The repository contains the following files:

- `*.html`: The HTML files for the website pages.
- `test.py`: A Python script for testing the website.
- `README.md`: This file.

## Setup

No setup is required. The website can be run by opening the HTML files directly in a web browser.

## Usage

To view the website, open any of the `.html` files in a web browser.

## Testing

To run the tests, execute the following command:

```bash
python3 test.py
```
